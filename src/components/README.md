# PAM AI

PAM AI is a premium fintech MVP focused on hypothetical financial decision modeling instead of backward-looking budgeting.

## What is included

- Landing page with product positioning and mockups
- Dashboard with realistic financial snapshot data
- Scenario Lab with mock calculations and side-by-side comparisons
- AI-style chat interface with rule-based projection responses
- Insights panel with warnings, opportunities, and levers

## Run locally

1. `cd /Users/iwillfixthis/Documents/New\ project/pam-ai`
2. `npm start`
3. Open `http://127.0.0.1:3040`

## Stack

- Native ES modules
- Responsive HTML/CSS/JS
- Small Node static server
